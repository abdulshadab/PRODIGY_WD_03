# ❌⭕ Tic-Tac-Toe Web Application

## 📖 Overview

This project demonstrates the development of an interactive Tic-Tac-Toe web application using HTML, CSS, and JavaScript. The game allows players to compete by placing markers on a 3×3 grid, aiming to align three markers in a row, column, or diagonal to win the game.

The application provides an engaging and responsive user experience with dynamic game state tracking and winner detection.

---

## ✨ Key Features

* 🎮 Interactive Tic-Tac-Toe gameplay
* 👥 Two-player mode
* 🤖 Optional AI opponent support
* 🏆 Automatic winner detection
* 🔄 Restart/reset game functionality
* 📱 Responsive and user-friendly interface

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the game board
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – Game logic and interactivity

## 🎯 Project Objective

The objective of this project is to:

* Implement game logic using JavaScript
* Handle user interactions and clicks
* Track and update game state dynamically
* Detect winning and draw conditions
* Build an engaging browser-based game

---

## 📸 Expected Behavior

* Players take turns placing markers (X and O)
* The application checks for winning conditions after each move
* Winner or draw message is displayed automatically
* Users can restart the game anytime
